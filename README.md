BootcampCLT - Productos API
Proyecto de API robusta desarrollada en .NET 8 siguiendo principios de Arquitectura Limpia y patrón CQRS con MediatR.
Diseñado para ser desplegado en contenedores mediante Docker y orquestado con Kubernetes, implementado con logs y métricas mediante SEQ desde kubernetes y Helm, y la viabilidad de realizar CI/CD.
